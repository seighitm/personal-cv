# Resume

## About
This project is a portfolio/resume demo website about my self. I created the website with React.js and Express.js. Enjoy!

## Run Project
```
> git clone https://github.com/seighitm/personal-cv.git
> cd personal-cv
```
### Client
```
> cd client
> npm install
> npm run start
```
### Server
```
> cd server
> npm install
> npm run dev
```
